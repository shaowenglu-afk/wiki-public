# 汽车金融 AI 建设方向（2026 上半年专题）

> 基于 11 篇微信公众号一手素材（2026-03 至 2026-06），梳理汽车金融机构 AI / 智能体建设的**真实路径、争议点与监管边界**。本页是 Wiki 在"行业垂直应用"维度的第一个深度概念页。

## 一句话总览

2026 年汽车金融行业正在经历**三重压力下的 AI 自救**：
1. 市场端**悲观情绪**（[[sources/55-悲观的汽车金融市场AI能救得了吗.md|FinTech 炼金术 2026-03]]）+ 比亚迪定价压力（[[sources/63-从比亚迪大唐定价策略看汽车金融定价趋势]]）
2. 监管端**频频踩红线**（[[sources/57-邮储银行因汽车金融业务风控漏洞被罚40万21名责任人遭追责]] / [[sources/60-从浦发到邮储工行汽车金融业务为何频踩监管红线]]）
3. 技术端**车企 + Agent 主导新战场**（[[sources/53-AI-Agent正在进入汽车新能源车企真正的AI战争才刚刚开始]] / [[sources/51-调研丨汽车智驾与AI-Agent融合趋势交流八千字深访]]）

## 核心议题

### 1. 主战场迁移：从银行车贷 → 车企生态 Agent

- **传统模式**：浦发 / 邮储 / 工行做车贷批发，靠零售网点 + 风控模型
- **新模式**：[[sources/53-AI-Agent正在进入汽车新能源车企真正的AI战争才刚刚开始|新能源车企真正的 AI 战争]] —— 主机厂自建 Agent + 金融科技，把车贷、保险、保养整合进车机
- **深度调研** [[sources/51-调研丨汽车智驾与AI-Agent融合趋势交流八千字深访|车市拾年 八千字]]：智驾 Agent 与车辆金融 Agent 走向融合
- **新业态**：[[sources/61-汽车金融遇上具身机器人下一个万亿赛道的融合与机遇|汽车金融遇上具身机器人]] —— 万亿赛道（投机性较强）

- [[sources/55-悲观的汽车金融市场AI能救得了吗\|FinTech 炼金术]] —— "AI 救不了"的悲观论调，与车企 Agent 乐观派形成对照

## 关键观察 / 跨文章共识

1. **车企 + Agent 是主战场**，银行系车贷正在边缘化
2. **风控 + 供应链** 是 AI 当下能落地的最大价值点（修远风控研社案例：效率 +100%、不良率 -？%）
3. **合规与监管**：2026 全行业被监管点名，合规风控反成核心竞争力
4. **悲观与乐观并存**：宏观市场悲观，但技术与车企生态乐观
