# Agent 可观测性工具爆发！五大开源项目让"跑飞"的 AI 不再裸奔

## TL;DR
**5 大开源 Agent 可观测性项目**（2026 年涌现）—— 给"裸奔"的 AI Agent 加上追踪、评估、调试能力。

## 推断的 5 大项目（业内常见）
- **Langfuse** —— LLM 应用追踪 + 评估
- **OpenLLMetry** —— OTel 扩展（Traceloop 出品）
- **Helicone** —— LLM 网关 + 可观测
- **Phoenix** （Arize AI 开源）—— LLM 评估 + 追踪
- **Langsmith** （LangChain 配套）—— 闭源但有开源对手

## 关键论点
- Agent 不可观测 = **生产事故隐患**
- 开源工具已经能覆盖 80% 场景，**商业产品只在企业治理 / 合规上有溢价**
- **OTel 是事实标准** —— 选项目优先看是否 OTel 兼容
