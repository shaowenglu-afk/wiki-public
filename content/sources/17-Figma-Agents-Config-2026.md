# Figma deep dive：Agents（Config 2026）

> Figma 官方 · Config 2026 · 2026-06-26 上传 · 27:21 · 字幕已抓取。

## TL;DR
Figma 把 Agent 定位为 **"canvas 上的协作者"** 而非又一个工具。具备 web search / live connectors / **teachable skills** / 上下文感知生成。Session 主线是 **如何与 Agent 协作**：什么交给它、什么留给自己、如何让创意直觉留在驾驶位。

## 四大能力
| 能力 | 含义 |
|---|---|
| Web Search | 主动联网取最新信息 |
| Live Connectors | 实时接入外部数据/服务 |
| **Teachable Skills** | 用户教授新技能 |
| 上下文感知生成 | 理解当前设计文件状态 |

## 关键命题
"很多 AI 讨论聚焦底层技术，**但本场聚焦用它建造的人，以及他们如何把它带进自己的创意流程**"。
