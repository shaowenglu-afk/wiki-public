# Portkey — AIOps Platform for Enterprises 2026 Guide

## TL;DR
**为 LLM / AI Agent 工作负载构建的 AIOps 平台应具备 5 大能力**：
1. **统一可观测性** —— 所有模型 / 所有 Agent 一个仪表盘
2. **路由韧性** —— 自动 fallback、retry、断路器
3. **护栏（Guardrails）** —— 内容过滤、PII 脱敏、合规检查
4. **Prompt 管理** —— 版本、A/B、模板
5. **治理与成本控制** —— Token 配额、预算告警、审计

## 关键论点
**LLMOps ≠ 传统 AIOps** —— 多模型路由 + Prompt 资产管理 + Token 成本是新维度，必须有专门平台。
