# Karpathy：从 Vibe Coding 到 Agentic Engineering（Sequoia AI Ascent 2026）

> YouTube · Sequoia Capital 频道 · 2026-04-29 上传 · 29:49 · 135.7 万播放。
> 主讲：Andrej Karpathy（OpenAI 联合创始人、前 Tesla AI 负责人、Eureka Labs 创始人）

## TL;DR
Karpathy 一年前造了 "**vibe coding**" 这个词，现在说**自己作为程序员从未感觉如此落后**。他把演讲组织为：Software 3.0 → Agent 作为 installer → jagged skills → 可验证性 → **从 Vibe Coding 进化到 Agentic Engineering**。核心金句："**你可以外包思考，但永远无法外包理解**"；"**LLM 不是动物，是幽灵**"。

## 八个核心论点
1. **作为程序员感觉落后** — 工具迭代太快，所有人都跟不上
2. **Software 3.0** — 用自然语言对 LLM 编程（1.0 = 手写代码、2.0 = 训练权重）
3. **Agent 作为 installer** — Agent 是软件世界的"安装程序"
4. **Menu Gen vs Raw Prompts** — 从"和模型聊天"转向"和工程系统对话"
5. **Jagged skills（参差能力）** — LLM 能力呈锯齿状，必须靠可验证性兜底
6. **Agentic Engineering** — vibe coding 是试运气，Agentic Engineering 是把"试"做成系统
7. **LLM 是幽灵** — 不要拟人化，是从人类文字中召唤出的概率性实体
8. **外包思考 ≠ 外包理解** — 理解必须自己保留

## 关键金句
- "I've never felt more behind as a programmer."
- "LLMs are not animals, they're ghosts."
- "**You can outsource your thinking but never your understanding.**"
