# CBA — DevOps Agent helps 2am on-call engineers

## TL;DR
澳洲联邦银行 (CBA) AWS Summit Sydney 披露：AWS frontier AI Agent 帮 on-call 工程师做事件投查。

**关键数据**：根因识别从 **60-180 分钟降到 30-50 分钟**。

**关键洞察**：消除"冷启动死时间"——告警进 PagerDuty 时 Agent 同步开始投查，工程师笔记本起来时直接看到投查总结 + 根因 + 补救建议。

**金句**："Nobody is at their best at two o'clock in the morning when you get a phone call asking why core banking is down."
