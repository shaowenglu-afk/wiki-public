# 腾讯云 — 企业级 Agent AI Native 架构设计与实践（完整版）

## TL;DR
ITPUB 中国系统架构师大会主题分享完整版。**5 个核心观点**：
1. **没有成熟的智能体架构，只有不断精进的智能体架构**
2. **EDD（Eval-Driven Development）** 是 Agent 时代新工程化范式
3. 企业未来朝 **APaaS（Agent Platform as a Service）**
4. **Markdown 文档即代码** —— 从 Vibe Coding 转向 AI 规范化编程
5. **现在企业级 AI Native 架构跑不出 7+2 结构**

## 关键公式
> **Agent = Model + Harness**
> 模型增强会蚕食 Harness——每次换模型，Harness 必须重评

## 关键论点
- **AI 集成 vs AI Native**：加 AI vs 用 AI 重构系统
- **3 次范式转变**：上下文 → 驾驭工程 / 单体 → 多 Agent / 项目级 → 平台级
- **4 层记忆体系**：工作 / 情景 / 语义 / 程序性
- **Agentic RAG** 替代 Naive RAG
