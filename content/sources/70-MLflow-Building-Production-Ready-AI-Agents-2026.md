# MLflow — Building Production-Ready AI Agents in 2026

## TL;DR
**Monolithic Agent 是陷阱**——原型快、生产脆。生产正确答案 = **微服务式多 Agent**。**Google Agent Bake-Off 实证**：处理时间 **1 小时 → 10 分钟**。**Microsoft AGT v3.6.0** 加入 **privilege rings + kill switches**——治理是基础设施，不是软护栏。**4 框架对比表**：AGT / Google ADK / LangGraph / MLflow Agent Platform。

## 关键论点
- 微服务式 Agent 让单个 sub-agent 可独立替换升级——**模型每几个月被替换时极其重要**
- Privilege Rings + Kill Switches 是 2026 治理新关键词
