# AI 运维进化拐点：可版本化运维 Skills

## TL;DR
**核心反共识**：比大模型更重要的是**把人的经验沉淀成可版本化、可复用、可迭代的运维 Skills**（类似 Kubernetes CRD）。**Skill = 可执行单元 + Prompt + 工具 + 验收标准 + 历史样本**。运维经验 = 可版本化数字资产。
