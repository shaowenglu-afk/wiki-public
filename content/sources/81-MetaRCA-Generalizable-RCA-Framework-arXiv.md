# arXiv 2603.02032 — MetaRCA: Generalizable RCA Framework

## TL;DR
**Meta Causal Graph (MCG)** 作为离线知识基础 + LLM + 历史故障 + 可观测性数据**三源融合**。

## 数据
- 252 公开故障 + 59 生产故障达到 **SOTA**
- **服务级 +29pp，指标级 +48pp**
- **跨系统泛化保持 >80% 准确率**（跨域 generalizability 是 RCA 长期挑战）

## 关键贡献
解决了 RCA 系统**只能在训练系统上工作**的痛点 —— MetaRCA 用 Causal Graph 抽象出业务无关的因果模式，可迁移。
