# n8n

本质是**自动化工作流引擎**，AI 只是它的能力之一。集成节点数量天花板。

## 一句话定义
600+ 节点的可视化自动化平台，擅长把各种 SaaS 工具串起来（邮件→提取信息→写表格→发通知）。

## 核心特征
- 🎯 **定位**：通用自动化（不是 AI 原生）
- 🟢 **优势**：
  - **600+ 集成节点**（远超 [[Coze]]、[[Dify]]）
  - 工作流编排最灵活
  - JavaScript 代码节点（复杂数据转换）
  - 企业微信 / 飞书原生支持
  - 支持私有部署 / SaaS / 混合部署
- 🔴 **限制**：
  - 学习曲线最陡，节点多连线密，初学者易晕
  - **RAG 需自己搭**（无内置）
  - 多轮对话管理需自己实现
  - AI 原生能力弱于 [[Dify]]

## 上手难度
⭐⭐⭐ 最高。复杂工作流需写 JavaScript 做数据转换，适合有编程基础的技术人员。

## 典型场景
- 跨系统自动化
- 数据处理流水线
- 已有 IT 基础设施的团队
- 与 [[Dify]] / [[Coze]] 组合做外围自动化、通知、定时任务
