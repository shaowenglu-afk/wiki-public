# OpenClaw

Peter Steinberger 主导构建的 CLI / Skills 生态系统，目标是把"个人 AI Agent"接入日历 / 邮件 / 推文 / 银行 / 谷歌全家桶等一切个人数字基础设施。

## 一句话定义
不是单一应用，而是一组围绕"用 CLI + Skills 构建私人 Agent"的开源工具集合。

## 关键事实
- 创始人 **Peter Steinberger** 已加入 OpenAI
- **很大程度上是用 [[Codex]] 开发的**
- Peter 是 [[Codex]] 重度用户：曾在 3 块显示器上开 18 个终端
- Peter 在 2025 年构建了 **40+ 个开源项目**，本质都在为同一愿景铺路：「我需要一个 CLI 来访问我的日历 / 推文 / Gmail」
- 这些工具最终演化为 OpenClaw 生态，并预示了今天 Coding Agents 通过 **Skills + CLI 工具** 工作的方式

## 在 OpenAI 的角色
帮助构建"**集成在 ChatGPT 中的下一代私人 Agent**"（访谈中 Romain 提到，细节暂未披露）。
