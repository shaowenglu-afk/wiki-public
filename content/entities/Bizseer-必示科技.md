# 必示科技 Bizseer

## 一句话
**国内 AIOps 平台层第一梯队代表** —— 清华裴丹教授孵化，金融 + 运营商 + 大型央企客户为主。

## 核心特征
- **背景**：清华大学计算机系裴丹教授孵化（学术派）
- **融资**：A+ 轮（高榕资本 + 东方富海）数千万人民币（2026-05）
- **客户**：国有大行 + 股份制银行 + 三大运营商 + 大型央企
- **核心能力**：
  - KPI 时序异常检测（学术派强项）
  - 告警收敛
  - RCA（根因分析）
  - 容量预测

## 国内 AIOps 厂商格局对照

| 公司 | 背景 | 客户重点 |
|---|---|---|
| **必示科技** | 清华裴丹团队 | 金融 + 运营商 |
| **云智慧** | 较早期 | 金融 + 政府 |
| **博睿数据** | 上市 | 互联网 + 金融 |
| **基调听云** | 上市 | 各行业 |

## 国内 vs 国外定位

**国内厂商**偏 **AIOps（事前去噪/异常检测）**——传统 ML 派
**国外厂商**偏 **AI SRE（事后调查）**——LLM Agent 派

两条路 2026 上半年正在融合。
