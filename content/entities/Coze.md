# Coze（扣子）

字节跳动出品的 AI Bot 开发平台，**门槛最低**的对话机器人构建工具。

## 一句话定义
拖拽式可视化界面，选模板 + 配 Prompt + 接插件即可上线 Bot，非技术人员也能用。

## 核心特征
- 🎯 **定位**：对话机器人 / Agent，不是通用工作流引擎
- 🟢 **优势**：
  - 学习曲线最低，零代码
  - 内置 RAG 知识库与多轮对话管理
  - 插件市场（Agent 工具调用）
  - 主流模型接入
- 🔴 **限制**：
  - 不支持私有部署，数据在云端 → 合规敏感企业不适用
  - SaaS / 数据库 / 企业微信 / 飞书集成有限，需插件补充
  - 无代码节点

## 适用场景
- 快速对话机器人原型
- 非技术团队的客服 / 运营 Bot
- 上线前的概念验证（之后可迁 [[Dify]] 重建）

## 常见组合
- **Coze + [[n8n]]**：前端 Bot + 后端数据/业务逻辑
- **Coze → [[Dify]]**：快速验证后迁至支持私有部署的平台
