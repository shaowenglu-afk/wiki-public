# Obsidian

本知识库使用的本地优先 Markdown 笔记软件。

## 一句话定义
基于本地 Markdown 文件的"第二大脑"工具，通过 Wiki-link、插件、Dataview 等机制构建知识网络。

## 配置同步要点
所有配置 / 插件 / 主题都保存在 vault 根目录的**隐藏文件夹 `.obsidian`**。

新建仓库后想恢复配置：
1. 复制旧 vault 的 `.obsidian` 文件夹
2. 粘贴到新 vault 根目录，提示已存在 → 替换
3. 重启 Obsidian

显示隐藏文件：
- macOS：`Cmd + Shift + .`
- Windows：参见 [[sources/04-Obsidian-新仓库配置同步]] 原文
