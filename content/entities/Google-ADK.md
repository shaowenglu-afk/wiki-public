# Google ADK (Agent Development Kit)

## 一句话
Google 出品的 **Agent Development Kit** —— 强项是 **Supervisor 层模式 + 多 Agent 性能流水线**，是 **Agent Bake-Off** 实证 1h→10min 的载体框架。

## 核心特征
- **Supervisor 层模式** —— 内置编排器 + worker 模式
- **Bake-Off Evaluation 模式** —— Google 独有的评估方法论
- **多 Agent 协作高可组合**

## 局限
- **原生 Sandbox 有限**（vs Microsoft AGT 的容器化）

## 适用场景
- **性能关键、多 Agent 流水线**
- 需要 Bake-Off 风格评估

## 实证案例
- **Google Agent Bake-Off**：处理时间从 **1 小时 → 10 分钟**（[[sources/70-MLflow-Building-Production-Ready-AI-Agents-2026]]）
