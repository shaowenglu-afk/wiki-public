# AI SRE: The Converging Operating Model

## 中文 TL;DR
**核心论点**：AI SRE 是**3 个传统领域融合**的新运营模型：

```
      Cloud Engineering（云工程）
              ↓
    SRE（站点可靠性工程）  ←→  AI Engineering（AI 工程）
              ↘    ↙
            AI SRE ⭐
```

## 3 领域融合的能力栈

| 来自 | 必备技能 |
|---|---|
| **Cloud Engineering** | K8s / IaC / 微服务 / 多云 |
| **SRE** | SLO / 错误预算 / Postmortem / Chaos |
| **AI Engineering** | LLM / Agent / Prompt / RAG / MCP |

## 关键论点
- **没人天生跨 3 领域** —— 必须主动补
- **SRE 转型 AI SRE 最容易** —— 已有 SLO + Postmortem 思维
- **AI Engineer 转 AI SRE 最难** —— 不懂生产真实复杂度
- "**未来 2 年招聘市场最稀缺的就是 AI SRE**"
