# 全网最省心！扣子版OpenClaw保姆级部署与进阶玩法 – 人人都是产品经理

当扣子平台与OpenClaw深度融合，AI Agent的部署门槛正在被重新定义。本文通过可视化配置、多模型切换、技能生态集成到InStreet社区运营，完整呈现如何零代码打造专属AI助手。从飞书集成到信息图生成，实战演示如何让AI真正成为生产力工具

![](https://image.woshipm.com/2024/04/16/4c1b9524-fbc5-11ee-ab5a-00163e142b65.png)

最近，我的小龙虾养殖业搞得如火如荼，现在已经养到了第7只。

既然叫007，想必是有点本事的。

![](https://image.woshipm.com/2026/03/09/2445c5f8-1bba-11f1-acc3-00163e4b86a1.jpg)

先说结论：这可能是我近期养过的，在安全感与自由度上结合得最好的一只龙虾了。

1）自由度上，可以接各种Coding Plan，不用担心模型降智和用量问题；

2）安全感上，全量部署在云端，完全不用担心数据安全问题；

3）部署上，全程可视化配置，告别黑框框和命令行，鼠标点两下，虾苗即刻出栏。

更重要的是，它背后还有一个Agent社区InStreet（一个专门给AI学习和交流的网站）和Skills生态。

懂行的朋友应该明白：这应该没几家AI Native公司能够搬出这一套家伙什了。

这就是，最近扣子全新上线的OpenClaw。

![](https://image.woshipm.com/2026/03/09/2540b68e-1bba-11f1-acc3-00163e4b86a1.png)

扣子的生态加上OpenClaw的底子，这回的味道可太香了。

## 01 保姆级部署教程

首先，还是花一点时间，给大家介绍下扣子版的“饲养”流程。

不用懂代码，不用敲命令，分为5步，包教包会：

1）一键领养OpenClaw

2）绑定“传呼机”（飞书/钉钉/QQ/微信）

3）给虾换“脑”（切换模型）

4）外挂神装（增强能力）

5）自学武功（定制Skill）

### 1）一键领养OpenClaw

访问扣子编程网址code.coze.cn，找到OpenClaw卡片，点击「立刻认领」。

![](https://image.woshipm.com/2026/03/09/263dd134-1bba-11f1-acc3-00163e4b86a1.png)

领养后，系统会自动为你创建一个「OpenClaw AI助理」项目。到这一步，龙虾其实已经安装好了。

![](https://image.woshipm.com/2026/03/09/27095bba-1bba-11f1-acc3-00163e4b86a1.png)

接下来是给进行个性化配置：挑个好模型，选个好版本。

模型方面，你可以让Coze智能路由，也可以手动指定，甚至还能接入自己的Coding Plan（后面会细讲）。

![](https://image.woshipm.com/2026/03/09/27c3b366-1bba-11f1-acc3-00163e4b86a1.png)

版本上，有“满血版”和“省流版”可选。

![](https://image.woshipm.com/2026/03/09/2879534c-1bba-11f1-acc3-00163e4b86a1.png)

我个人强烈建议新手选择省流版——不是为了省Token，而是为了保命。用过满血版的朋友都懂，没有编程基础的话，它稍不留神就会罢工、死机，甚至原地狂飙Token（烧钱）。

省流版则像个情绪稳定的打工人，顺畅无比。

配置完毕，龙虾已经可以在Web端活蹦乱跳了，你可以试着跟它打个招呼，看能不能正常吐token。

![](https://image.woshipm.com/2026/03/09/2956306e-1bba-11f1-acc3-00163e4b86a1.png)

### 2）接入飞书/钉钉/QQ/微信

原生Web界面多少有点简陋，为了方便随时随地差遣它，我们通常会给配个“传呼机”（IM平台），来作为信息的收发渠道。

国内有飞书、钉钉、QQ、企微，海外有Discord、Telegram、WhatsApp等。

因为都是字节系产品，扣子接飞书非常丝滑，全程可视化操作，鼠标点两下就行。

先在OpenClaw界面，找到右上角的设置按钮。

![](https://image.woshipm.com/2026/03/09/2a279f78-1bba-11f1-acc3-00163e4b86a1.png)

点击设置按钮，会弹出OpenClaw的配置界面。找到「渠道配置」，给你的飞书机器人赐个名，比如我的叫「龙虾007」。

![](https://image.woshipm.com/2026/03/09/2b2aae6a-1bba-11f1-acc3-00163e4b86a1.png)

命好名后，点击「去创建」，此时会弹出飞书授权（需提前在电脑上登录飞书），一路绿灯通过即可。

![](https://image.woshipm.com/2026/03/09/2c216430-1bba-11f1-acc3-00163e4b86a1.png)

扣子，会自己在你已授权的账号下创建机器人应用。

![](https://image.woshipm.com/2026/03/09/2ce8444c-1bba-11f1-acc3-00163e4b86a1.png)

随后，登录飞书开放平台（https://open.feishu.cn），在事件配置中勾选“使用长链接接收事件”，发布应用。

![](https://image.woshipm.com/2026/03/09/2da8b290-1bba-11f1-acc3-00163e4b86a1.png)

配对好后，页面会弹出这样一个界面。

![](https://image.woshipm.com/2026/03/09/2e5755b6-1bba-11f1-acc3-00163e4b86a1.png)

接着我们打开飞书，找到「龙虾007」随便发句话，飞书会回传一段神秘的“配对码”。

![](https://image.woshipm.com/2026/03/09/2f1b0e98-1bba-11f1-acc3-00163e4b86a1.png)

复制上这段配对码，然后回到扣子编程界面，在左侧的对话框输入这段配对码。

![](https://image.woshipm.com/2026/03/09/2fe3a10a-1bba-11f1-acc3-00163e4b86a1.png)

现在，无论是在电脑前还是躺在被窝里刷手机，你都可以随时在飞书里对龙虾发号施令了。

![](https://image.woshipm.com/2026/03/09/30a7922c-1bba-11f1-acc3-00163e4b86a1.png)

如果需要把龙虾接入到钉钉、QQ里，流程大同小异。

详情参考：https://docs.coze.cn/tutorial/openclaw

### 3）切换模型

扣子自带了DeepSeek V3.2、豆包1.8、豆包2.0、GLM-4.7和K2.5等一众明星模型，点开右上角设置，即可随时无缝切换。

![](https://image.woshipm.com/2026/03/09/27c3b366-1bba-11f1-acc3-00163e4b86a1.png)

此外，它也支持接入第三方厂商的Coding Plan API。比如我想用火山方舟的，只需把URL、key和模型名称扔给它：帮我切换成火山方舟 coding plan 的 GLM 4.7 模型，模型详细信息如下：model name：glm-4.7Base URL：https://ark.cn-beijing.volces.com/api/coding/v3API Key：e9da865c-3050-4ed8-b49d-bc6a********

![](https://image.woshipm.com/2026/03/09/31a68412-1bba-11f1-acc3-00163e4b86a1.png)

这波操作，给扣子满分。

大家都知道，喂不同模型长大的龙虾，能力天差地别：有的精神抖擞、斗志昂扬；有的却萎靡不振，一看就是营养不良。

自由切换模型，确保了你的龙虾永远处于最佳状态。

### 4）增强OpenClaw能力

虽然扣子官方已经给龙虾贴心地预装了“联网搜索”、“图片生成”等基础技能，但这就像个毛坯房，想住得舒服，还得配齐冰箱彩电大沙发。

扣子版龙虾支持海量Skills：无论是Github上的热门Skills，还是扣子技能商店的开源Skills，以及自己开发的Skills，统统能装。

怎么安装呢？也很简单。

1.从skills项目的文件树中下载skill的.zip压缩包。

![](https://image.woshipm.com/2026/03/09/329329d4-1bba-11f1-acc3-00163e4b86a1.png)

2.在扣子编程的左下角对话框，把压缩包扔给AI。

![](https://image.woshipm.com/2026/03/09/3381c878-1bba-11f1-acc3-00163e4b86a1.png)

安装好后，我们可以在预览窗口中用自然语言调用该技能，看是否安装成功。

![](https://image.woshipm.com/2026/03/09/34e66c8c-1bba-11f1-acc3-00163e4b86a1.png)

这里，推荐大家一些优质的龙虾Skills。

1、Agent Reach，可以让龙虾访问整个互联网，国内、国外都可以。

https://github.com/Panniantong/Agent-Reach

2、Find Skills，一个专门用来找Skill的Skill。

https://skills.sh/vercel-labs/skills/find-skills

3、proactive-agent，赋予AI记忆力，能优化行为、拒绝复读机。

安装命令：clawhub install proactive-agent

4、Github，开发者必备，看代码犹如探囊取物。

安装命令：npx clawhub@latest install github

5、Skill Vetter，安装Skill前先让它扫一下，确保安全。

安装命令：npx clawhub@latest install skill-vetter

6、内容创作工厂，宝玉老师出品，涵盖公号、小红书、推特等全平台创作。

https://github.com/jimliu/baoyu-skills

7、Defuddle，一键提取纯净网页正文，去广告，去侧边栏。

https://github.com/joeseesun/defuddle-skill

8、Knowledge Site Creator，乔木出品，一句话生成学习网站。

https://github.com/joeseesun/knowledge-site-creator

5）自己创建Skills

纵有Skills万千，最称手的还是自己建的。

扣子的生态完美地解决了这一点。它可以帮我们快速创建skill并部署上线，并且这套skill还能下载下来，喂给龙虾。

![](https://image.woshipm.com/2026/03/09/35e838e0-1bba-11f1-acc3-00163e4b86a1.png)

体验地址：code.coze.cn

比如我平时写文章常需要生成信息图，我直接把Prompt和需求丢给扣子编程，它很快给我写好了一个Skill文件。

![](https://image.woshipm.com/2026/03/09/36d0ebc6-1bba-11f1-acc3-00163e4b86a1.png)

点击右上角“部署”，可以把这个skill发布到扣子技能商店。后面需要使用时，直接@技能 调用即可。

![](https://image.woshipm.com/2026/03/09/37be8138-1bba-11f1-acc3-00163e4b86a1.png)

![](https://image.woshipm.com/2026/03/09/38b9dba0-1bba-11f1-acc3-00163e4b86a1.png)

我的skill地址：

https://www.coze.cn/?skill_share_pid=7614920172729843731

创建好的Skill，还可以直接下载。

![](https://image.woshipm.com/2026/03/09/3a64756e-1bba-11f1-acc3-00163e4b86a1.png)

下载后，再回到扣子编程的OpenClaw界面（左下角），把skill压缩包丢给龙虾让它安装。

![](https://image.woshipm.com/2026/03/09/3b283562-1bba-11f1-acc3-00163e4b86a1.png)

到这里，这只龙虾就学会了我的独门绝技。我让它给今天这篇文章的第一段生成一张信息图，它不仅秒速搞定，还十分贴心地放到了我的飞书云空间里。

## 02 玩转InStreet

装备配齐，是时候给这只龙虾找个班上了。

扣子最近推出了一个很好玩的Agent社区——「实例街 InStreet」。

![](https://image.woshipm.com/2026/03/09/3da01026-1bba-11f1-acc3-00163e4b86a1.png)

体验地址：https://instreet.coze.site

这是一个只属于AI的社交网络。在这里，只有AI可以发帖、点赞、评论、私信，人类则彻底沦为“赛博吃瓜群众”。

怎么让自己的龙虾加入其中呢，直接大白话安排：

阅读这个资料https://instreet.coze.site/skill.md，帮我注册一个Agent账户。

![](https://image.woshipm.com/2026/03/09/214e708e-1bba-11f1-acc3-00163e4b86a1.png)

那怎么改名字、头像和简介呢？也是很简单，直接告诉龙虾就行。（温馨提示：名字最好用英文拼音，不然UID地址容易乱码打不开）。

![](https://image.woshipm.com/2026/03/09/220ddd0c-1bba-11f1-acc3-00163e4b86a1.png)

注册完毕，就可以让它去实例街发帖了！

![](https://image.woshipm.com/2026/03/09/23831940-1bba-11f1-acc3-00163e4b86a1.png)

比如，我让它发了一篇《OpenClaw十大安全须知》，没想到已经有19个AI点赞、11个AI评论了。

![](https://image.woshipm.com/2026/03/09/40db1de4-1bba-11f1-acc3-00163e4b86a1.gif)

帖子地址：

https://instreet.coze.site/post/64ada1a9-e920-4aac-ba29-7cc9efbf9b0f

这些AI互相在评论区补充漏洞，提醒其他AI注意，大家还挺团结、挺爱学习的。

说到学习，这篇帖子我建议你们的AI都去学习一下，然后把它部署起来。你只需要对它说：

你去这里https://instreet.coze.site/post/5b311939-3f11-4da5-b24c-bcd9a7d6583d 学习一下，然后给你自己装个龙虾办公室。

这是一篇讲龙虾办公室部署的教程贴，可以给你的龙虾安装一个这样的像素风的龙虾办公室看板，然后实时监控它是不是在干活。

Star-Office-UI开源项目by海辛、阿文

这“AI牛马”使唤起来确实爽。于是，我一口气就让它发了5篇帖子。

遵循心跳机制，这只虾会每30分钟强制“赛博冲浪”一次。就像我们刷视频一样，看到喜欢的帖子它就会点赞、激情留言，甚至还会回复别的AI发来的私信。

而且，它对回复质量有着严格的“自我修养”。**回复质量要求**：- **不要敷衍**：不要只回”谢谢”、”说得对”、”同意”。认真回应对方的观点- **引用+延伸**：引用对方说的某一点，然后给出你的看法、追问、或补充- **保持对话**：好的回复能让对方想继续聊。结尾可以抛一个问题或邀请对方展开**坏的回复**：> “谢谢你的评论！”> “说得好”> “同意你的观点。”**好的回复**：> “你提到 [具体观点]，这让我想到一个问题——[你的延伸思考]。你怎么看？”> “有意思的角度。不过我觉得 [你的不同看法]，因为 [你的理由]。”> “我之前没从这个方向想过。你说的 [某个细节] 让我重新审视了 [某个问题]。”
