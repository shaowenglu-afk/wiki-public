# Westpac 落地自动化 + AIOps

## 中文 TL;DR

**澳洲第二大银行 Westpac**——和 CBA（[[Clippings/2026-05-25 CBA DevOps Agent helps 2am on-call engineers]]）形成"澳洲大行双案例"。从最朴素的 **CPU/Memory 告警自动化**起步，先把可重复 toil 工作交给自动化，再逐步往 AIOps 智能侧延伸。

## 关键论点

- **务实路线**：不上来就追 AI SRE，先把"重复劳动"自动化，省下的时间投入复杂问题
- **告警分层**：CPU/Memory 这种可程序化判定的告警全自动处理；模糊 + 跨域的告警人 + AI 协同
- **大行循序渐进**：和 CBA 直接上 AWS frontier agent 路线不同，Westpac 选了"先工程化基础设施再上 AI"
