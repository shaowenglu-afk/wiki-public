# AIOps Skills Matrix 2026

## 中文 TL;DR

**AIOps 团队的核心角色集群 + 三级能力矩阵 + 转型路径**：

## AIOps 团队 4 大核心角色

1. **AIOps Engineer** —— 落地实施
2. **Data Engineer** —— telemetry 数据管道
3. **ML/AI Engineer** —— 模型 / Agent 工程化
4. **Site Reliability Architect** —— 整体架构 + SLO

## 三级能力矩阵

### Level 1: Foundational Practitioner（基础实践者）
- 理解传统 monitoring / alerting
- Prometheus / Grafana 基础
- 写 PromQL / 简单脚本

### Level 2: Applied AIOps Engineer（应用工程师）⭐ 主力
- **熟悉 LLM + Agent 框架**（LangChain / LangGraph）
- **会写 Prompt + 调用工具链**
- **可观测 Prompt / Tool / Trace / Token 4 维**
- 理解 RCA 自动化原理

### Level 3: Strategic AIOps Architect（架构师）
- **设计多 Agent 协作架构**
- 把握 build vs buy 决策
- 安全 + 合规 + 治理整体设计
- 跨团队推动

## 工具能力期望（4 大领域）

| 领域 | 工具 |
|---|---|
| **可观测 + 遥测** | OpenTelemetry / Grafana / Loki / Tempo / Datadog |
| **数据工程 + ML 基础** | Kafka / Spark / Python + PyTorch / Vector DB |
| **自动化 + 集成** | Terraform / Ansible / GitOps / MCP |
| **安全 + 治理** | OPA / Vault / IAM / Audit logging |

## 3 条转型路径

| 起点 | 终点 | 关键补课 |
|---|---|---|
| **SRE** → AIOps Engineer | LLM + Agent + RCA 自动化 |
| **Data Engineer** → AIOps Specialist | 实时遥测 + 故障域知识 |
| **Security Engineer** → AI-Driven SecOps | LLM 安全 + Prompt 注入防护 |

## 招聘常见陷阱

- ❌ 招"懂 ML 不懂运维"的人 —— 实际不知道 SLO / Postmortem
- ❌ 招"懂运维不懂 LLM"的人 —— 把 Agent 当 RPA 用
- ✅ 招"两边都有基础 + 愿意学"的人
