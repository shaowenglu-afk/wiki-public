# Why AI Skills for Cloud/DevOps Engineers Matter in 2026

## 中文 TL;DR

**核心论点**：Cloud / DevOps 工程师**不补 AI 技能 = 5 年内被淘汰**。

## 5 大必学技能

1. **LLM API 调用**（OpenAI / Anthropic / Bedrock）
2. **Prompt 工程**（不只是简单 Q&A）
3. **Agent 框架**（LangChain / LangGraph / AutoGen）
4. **RAG + Vector DB**（Pinecone / Weaviate / Chroma）
5. **MCP + 工具集成**

## 6 大实操项目（学习路径）

1. **自动化 PR 审查 Agent**
2. **K8s 故障自愈 Agent**
3. **日志智能分析**
4. **告警去噪 LLM 分类器**
5. **Postmortem 自动起草工具**
6. **多 Agent 协作平台 PoC**
