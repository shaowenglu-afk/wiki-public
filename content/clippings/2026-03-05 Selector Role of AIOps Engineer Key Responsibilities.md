# Role of an AIOps Engineer: Key Responsibilities

## 中文 TL;DR · 6 大核心职责

1. **数据管道** —— 整合 logs / metrics / traces 输入到 AI 系统
2. **模型训练 & 调优** —— 异常检测 / RCA 模型
3. **Agent 编排** —— 多 Agent 协作配置
4. **告警治理** —— 去噪 + 关联 + 升级路径
5. **Postmortem 自动化** —— Agent 起草 + 工程师审
6. **跨团队协作** —— 与 DevOps / Security / 业务团队对齐

## 关键技能要求
- **必备**：Python + SQL + 可观测栈 + LLM API
- **加分**：MLOps + LangGraph + Kafka + Vector DB
