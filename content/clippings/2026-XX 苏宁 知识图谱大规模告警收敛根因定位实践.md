# 苏宁基于知识图谱的大规模告警收敛和根因定位实践

## 中文 TL;DR

**国内大型零售/互联网集团苏宁** AIOps 工程化一手案例，跟[[Clippings/2026-06-25 AI Agent协作运维在新浪微博的落地与演进]]、[[Clippings/2026-06-23 大型集团IT运维智能体（AIOps Agent）自主故障诊断与自愈平台详细设计方案]] 形成"国内大厂 AIOps 实战集群"。

## 核心技术栈
- **知识图谱**：组件 → 服务 → 系统 → 业务的拓扑关系图
- **告警收敛**：基于图谱的关联分析，把数千告警合并成数十事件
- **根因定位**：图谱遍历 + 时序相关性分析

## 关键论点
- **国内电商高峰场景特殊**：双 11 + 618 流量是平时几十倍，告警放大效应巨大
- **知识图谱是 RCA 的关键基础设施** —— 比纯统计 ML 更可解释
- **vs LLM Agent**：知识图谱是结构化的，LLM Agent 是自适应的，**2026 趋势是两者结合**
