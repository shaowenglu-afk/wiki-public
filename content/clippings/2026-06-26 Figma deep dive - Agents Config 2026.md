## 视频简介（中译）

Figma Agent 正在改变设计师在 Figma 中的工作方式。它**不是又一个工具，而是 canvas 上的协作者**。具备：
- 联网搜索（web search）
- 实时连接器（live connectors）
- **可教学技能（teachable skills）**
- 上下文感知的生成引擎

它能帮助探索想法、自动化繁琐工作、把设计转成原型、并在设计环境内直接支持协作。

本场 Session 拆解最新 Agent 能力，**更重要的是讲：如何在设计中与 Agent 协作——什么交给它、什么留给自己、如何让你的创意直觉留在驾驶位**。

## 中文要点提炼（基于字幕）

### 1. 开场：来自产品和研究团队的视角
- 主持：Georgia（Figma 产品团队）、Angel（Figma 研究团队）
- 嘉宾：Tammy 和 Rodrigo（对 Figma agent 发布关键贡献）

### 2. 立场：协作者而非工具
"很多 AI 讨论聚焦在底层技术，但本场聚焦在**用它建造的人，以及他们如何把它带进自己的创意流程**。"

### 3. 四大能力组件
| 能力 | 含义 |
|---|---|
| Web Search | 主动联网获取最新信息 |
| Live Connectors | 实时接入外部数据/服务 |
| **Teachable Skills** | 用户可教授新技能（与 Anthropic 的 Agent Skills、OpenAI 的 Codex Skills 同源） |
| 上下文感知生成 | 理解当前设计文件状态 |

### 4. 应用场景
- 探索想法（idea exploration）
- 自动化繁琐工作
- 设计 → 原型转换
- 设计环境内直接协作

### 5. 设计原则：留在驾驶位
Session 主线**不是炫能力**，而是讲 **how to work with agents**：
- 什么交给 Agent（重复、机械、生成探索）
- 什么留给自己（关键创意决策、品味判断）
- 如何配置，让创意直觉留在驾驶位

## 与本知识库的呼应

- "**Skills**" 抽象的跨厂商收敛：Figma teachable skills · Codex Skills · OpenClaw CLI skills · Anthropic Agent Skills——都把"技能"作为 Agent 接入垂直工具生态的标准化单元
- "Agent = 协作者而非工具" 的定位与 Karpathy 视频里 "Software 3.0 / Agent as installer" 的思路同源
